# Questionaire – AI Interview Preparation Platform

Questionaire is an AI-powered interview preparation platform that generates role-specific interview questions using OpenAI GPT models.

Users can select a job role, experience level, and difficulty level to instantly generate a customized interview kit containing technical, behavioral, and HR questions along with expected answers and follow-up questions.

## Features

* AI-generated interview questions
* Technical, Behavioral, and HR categories
* Expected answers for each question
* Follow-up questions
* Difficulty levels (Easy, Medium, Hard)
* Experience-based customization
* MongoDB database integration
* Modern React UI
* Export interview kits
